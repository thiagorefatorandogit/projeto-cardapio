🍽️ Sabor Brasileiro — Website de Restaurante

Projeto de site responsivo para restaurante com foco em experiência visual, apresentação de cardápio e interação básica de carrinho.

Este projeto foi desenvolvido utilizando HTML, CSS e JavaScript puro (Vanilla JS) como prática de front-end e manipulação do DOM.

📸 Preview do Projeto

Interface moderna com:

Hero com slider automático
Cardápio interativo
Carrinho com contador
Layout responsivo
Footer completo com formulário de contato

🚀 Funcionalidades
🎬 Hero + Slider automático
Troca automática de imagens a cada 8 segundos
Efeito de fade + zoom nas imagens
Botão "Ver Cardápio" com scroll suave

📖 Cardápio interativo
Cards de produtos com imagem, descrição e preço
Botão Adicionar ao Carrinho
Feedback visual ao adicionar item

🛒 Carrinho (simulação)
Contador dinâmico no header
Atualização em tempo real via JavaScript

📱 Responsividade
Layout adaptado para mobile
Menu preparado para versão mobile
Grid responsivo de cards

🦶 Footer completo
Navegação rápida
Informações de contato
Ícones sociais
Formulário de mensagem
Ano atualizado automaticamente via JS
